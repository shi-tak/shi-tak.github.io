## はじめに

Rubyライブラリの開発にあたって、公開リポジトリの設定を調べています。

こちらは私のリポジトリの設定値のメモを記載します。

## プルリクエスト

### Settings > General

Automatically delete head branches: オン
マージ時にブランチを削除する。

### Settings > Access > Moderation Options

Code review limits: オン
プルリクエストのレビューや承認を実施できるユーザーを制限する。


## セキュリティ

### Settings > Security > Advanced Security

Advanced Security features help keep your repository secure and updated. By enabling these features, you're granting us permission to perform read-only analysis on your repository.

Private vulnerability reporting: Enable

Dependency graph: Enabled

Automatic dependency submission: Enable

Dependabot alerts: Enable

Dependabot rules: 1 rule enabled (default)

Dependabot security updates: Enable

Grouped security updates: Disable

Dependabot version updates: Enable

CodeQL analysis: default setup

Other tools: なし
